# Справочник по политикам Microsoft Intune

Используйте сведения в этом разделе для определения политики Microsoft Intune, необходимой для управления устройствами.

> Дополнительные сведения об использовании политик см. в разделе [Управление параметрами и компонентами на устройствах с помощью политик Microsoft Intune](manage-settings-and-features-on-your-devices-with-microsoft-intune-policies.md).


## Политики конфигурации Android

|Имя политики|Обеспечиваемые возможности|
|---------------|------------------------|
|**Пользовательская конфигурация (Android 4 и более поздние версии, Samsung KNOX Standard 4.0 и более поздние версии)**|Развертывание параметров OMA URI, таких как параметры Wi-Fi, которые можно использовать для управления функциями устройства. Это полезно в том случае, если необходимый параметр недоступен в политике конфигурации.<br /><br />Дополнительные сведения см. в разделе [Параметры политики Android в Microsoft Intune](android-policy-settings-in-microsoft-intune.md).|
|**Профиль электронной почты (Samsung KNOX Standard 4.0 и более поздние версии)**|Создание, развертывание и отслеживание параметров электронной почты Exchange ActiveSync на управляемых устройствах. Это позволяет пользователям получать доступ к корпоративной почте с собственных устройств без дополнительных настроек с их стороны.<br /><br />Дополнительные сведения см. в разделе [Настройка доступа к корпоративной электронной почте с помощью профилей электронной почты с Microsoft Intune](configure-access-to-corporate-email-using-email-profiles-with-microsoft-intune.md).|
|**Общая конфигурация (Android 4 и более поздние версии, Samsung KNOX Standard 4.0 и более поздние версии)**|Настройка параметров безопасности и функциональных параметров мобильных устройств.<br />Определение приложений, соответствующих или не соответствующих политике, и создание отчета при их использовании.<br />Настройка режима киоска, в котором можно заблокировать устройства и разрешить работу лишь определенных функций, например разрешить устройству запускать только одно приложение или отключить клавиши регулировки громкости.<br /><br />Дополнительные сведения см. в разделе [Параметры политики Android в Microsoft Intune](android-policy-settings-in-microsoft-intune.md).|
|**Профиль сертификата PKCS #12 (PFX) (Android 4 и более поздние версии)**|Используйте этот профиль для создания и развертывания параметров PFX для запросов на сертификаты устройств.<br /><br />Дополнительные сведения см. в разделе [Защита доступа к ресурсам с помощью профилей сертификатов в Microsoft Intune](secure-resource-access-with-certificate-profiles.md).|
|**Профиль сертификата SCEP (Android 4 и более поздней версии)**|Настройте сертификат SCEP, который можно использовать с сертификатом доверенного мобильно устройства для проверки подлинности мобильных устройств, чтобы разрешить им доступ к сетевым ресурсам, например настроенным профилями Wi-Fi и VPN.<br /><br />Дополнительные сведения см. в разделе [Защита доступа к ресурсам с помощью профилей сертификатов в Microsoft Intune](secure-resource-access-with-certificate-profiles.md).|
|**Профиль доверенных сертификатов (Android 4 и более поздней версии)**|Настройте сертификат доверенного мобильного устройства, который можно использовать для проверки подлинности мобильных устройств, чтобы они могли получить доступ к сетевым ресурсам, например настроенным профилями Wi-Fi и VPN.<br /><br />Дополнительные сведения см. в разделе [Защита доступа к ресурсам с помощью профилей сертификатов в Microsoft Intune](secure-resource-access-with-certificate-profiles.md).|
|**Профиль VPN (Android 4 и более поздней версии)**|Настройте и выполните развертывание параметров, которые предоставляют пользователям безопасный доступ к сети организации с мобильного устройства. Развертывание этих параметров упрощает для конечного пользователя подключение к рабочей учетной записи.<br /><br />Дополнительные сведения см. в разделе [VPN-подключения в Microsoft Intune.md](vpn-connections-in-microsoft-intune.md).|
|**Профиль Wi-Fi (Android 4 и более поздней версии)**|Настройте и выполните развертывание параметров беспроводной сети для пользователей в вашей организации. Развертывание этих параметров упрощает подключение к беспроводной сети для конечного пользователя.<br /><br />Дополнительные сведения см. в разделе [Подключения Wi-Fi в Microsoft Intune](wi-fi-connections-in-microsoft-intune.md).|
