# На устройстве не включено шифрование BitLocker

Организации требуется включить программу BitLocker, которая шифрует диск и защищает данные. Инструкции по включению BitLocker на устройствах Windows 10 см. в документе [Способы включения BitLocker на устройствах Windows 10](https://gallery.technet.microsoft.com/How-to-turn-on-BitLocker-34294d3d). Чтобы получить инструкции для других устройств, обратитесь к ИТ-администратору.
